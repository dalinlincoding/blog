## WHAT-什么是虚拟DOM
    在JavaScript中，可以很方便的使用JS对象实现DOM对应的结构，那么当DOM发生改变时，很容易知道那些内容发生了改变，然后去刷新对应的真实DOM，避免粗放式的DOM操作。

## HOW-如何实现虚拟DOM
  ### 通过JS对象表示虚拟DOM

  ```
    var vDOM = {
      oldV,
      newV,
      index,
      [{

        newV,
  ```
